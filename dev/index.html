SpatialEcology.jl

Community- and macro-ecological analysis in julia

Package Features

  • Easily load macroecological and community ecological datasets
  • Easy and cheap views allows analysis in a split-apply-combine framework
  • Plots recipes for all objects
  • Compatible with Diversity.jl via the EcoBase framework

The Quick start provides a tutorial with a quick overview to get you started with SpatialEcology.

The best way to see what the package can do is to work through some of the Example analyses using this package to recreate macroecological analyses from published papers.
